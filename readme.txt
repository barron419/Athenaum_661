Setup Instructions for Athenaum Media Library

1) Create MySQL Database 'athenaum' on localhost and allow access for user: phpuser, password: phpuserpw
    1a) Alternatively, modify config.php to use different schema & user.
2) Run setup.php
    2) Setup.php will create necessary tables and insert relevant information into tables to use/test Athenaum Media Library and automatically redirect to homepage is successful.
